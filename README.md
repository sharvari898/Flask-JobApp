Job Application Submission App (Flask)
A simple Flask-based job application web app that allows users to submit job application details, stores them in a SQLite database, and sends a confirmation email upon successful submission.

Features
Job application form
Stores applicant data in database
Sends confirmation email using Gmail SMTP
Flash messages for user feedback
Logging for debugging
SQLite database (lightweight & simple)

Tech Stack
Python
Flask
Flask-SQLAlchemy
Flask-Mail
SQLite
HTML / Jinja2
